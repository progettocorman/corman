Guida dell'installazione:
	1- Scaricare il progetto da questo link:
		https://github.com/progettocorman/corman
	2- Caricare il file di dump nel proprio database
	3- Modificare il file .env all'interno della cartella del progetto settando le impostazioni del proprio database correttamente
	4- Da prompt recarsi nella cartella scaricata 
	5- Da prompt eseguire php artisan serve
	6- Accedere al sito all'indirizzo localhost:8000
